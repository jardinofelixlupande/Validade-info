## 📋 Sobre o Projeto
Este é um projeto desenvolvido para praticar conceitos avançados de **Desenvolvimento Web Front-end**, focando em organização de código, modularização e lógica de programação.

## 🛠️ Tecnologias Utilizadas
*   **HTML5:** Estrutura semântica das páginas.
*   **CSS3:** Estilização moderna utilizando **Flexbox** e **CSS Grid**.
*   **JavaScript (ES6+):** Lógica dinâmica, manipulação do DOM e armazenamento local.
*   **Git & GitHub:** Controle de versão.
